# AI评论回复助手

一款AI驱动的商家评论个性化回复工具，帮助生活服务商家高效回复美团、大众点评、口碑等多平台顾客评论。

## 功能特点

- **经营信息自由配置**：支持品牌名称、经营品类、品牌特色的自由文本输入
- **智能回复生成**：基于阿里云百炼大模型，结合商家特色生成个性化回复
- **多种语气选择**：活泼/专业/幽默三种回复语气
- **字数档位可选**：100/200/300/400字四种字数档位
- **无限重新生成**：不满意可随时重新生成，获取差异化回复
- **直接编辑修改**：生成内容可直接编辑，微调后可复制使用
- **一键复制**：快速复制到剪贴板，前往平台发布
- **无历史记录**：退出即清空，保护商家隐私
- **响应式设计**：完美适配手机端和电脑端

## 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 环境变量配置

在 `src/services/aiApi.js` 中配置您的 API Key：

```javascript
const CONFIG = {
  apiKey: 'YOUR_API_KEY',  // 替换为您的阿里云百炼 API Key
  appId: 'YOUR_APP_ID'      // 替换为您的智能体应用 ID
}
```

## 部署

### Vercel 部署

1. 将代码推送到 GitHub 仓库
2. 访问 [Vercel](https://vercel.com) 并导入项目
3. Vercel 会自动检测 Vite 项目并进行构建部署
4. 部署完成后获得可访问的 URL

### Netlify 部署

1. 将代码推送到 GitHub 仓库
2. 访问 [Netlify](https://netlify.com) 并导入项目
3. 设置构建命令为 `npm run build`，发布目录为 `dist`
4. 部署完成后获得可访问的 URL

## 使用流程

1. 首次使用进入「经营信息设置」填写品牌信息
2. 复制多平台顾客评论
3. 粘贴到评论输入框
4. 选择回复语气和字数档位
5. 点击「生成回复」
6. 满意则一键复制，不满意可重新生成或直接编辑

## 技术栈
